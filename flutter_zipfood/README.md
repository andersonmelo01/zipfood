# zipfood_app

Projeto Flutter inicial para o app Zipfood.

## Como iniciar

1. Certifique-se de ter o Flutter instalado: https://docs.flutter.dev/get-started/install
2. No terminal, navegue até esta pasta:
   cd flutter_zipfood
3. Execute:
   flutter pub get
   flutter run

## Estrutura inicial
- lib/main.dart
- pubspec.yaml
- android/
- ios/
- web/

Adicione o restante dos arquivos com o comando:
flutter create .
